== Description ==

Ushbu plagin TinyMCE tahrir oynasida o'zbek tilidagi matn imlosini tekshirish imkonini beradi. Plagin matn imlosini tekshirishda http://imlo.insof.uz xizmati API tizimidan foydalanadi.

== Installation ==

1. Plagin fayllarini `/wp-content/plugins/imlo` bo'limiga yuklang, yoki WordPress plagin o'rnatish sahifasi orqali saytingizga qo'shing.
2. WordPressning 'Plaginlar' sahifasini ochib, plaginni faollashtiring.
3. Maqolani tahrir qilish oynasiga o'ting. TinyMCE tahrir oynasida yangi tugma qo'shilganini ko'rasiz. Shu tugma tahrir oynasidagi matnni imlo xatolariga tekshiradi.
